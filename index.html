jak zacząć html rozkład grup grupy elementy grupy elementy interaktywne

jak zacząć index html

index html trzeba zacząć do napisania tego jak ma wyglądać zakładka strony u góry przeglądarki a to skrypt do tego




    <!DOCTYPE html>    <!-- tu jest poinformowanie skryptu że to jest dokument html -->

    <html lang="...">                  <!-- tu jest grupa html w której pisze się całą strone (i poinformowanie skryptu o tym po jakiemu jest nasza strona) -->
    
    <head>   <!-- grupa head nie wpływa na strone ale zakładke strony u góry przeglądarki -->
    
        <meta charset="UTF-8">     <!--  specifikacja tekstu narazie nieważne -->
        <meta name="viewport" content="width=device-width, initial-scale=1.0">      
        <title>...</title>               <!-- nazwa strony u góry przeglądarki  -->  
        <link rel="icon" href="favicon.png" type="image/...">     <!--ikonka strony w type dajemy typ obrazu -->
        
    </head> <!--zakączenie grupy head tak jak kończymy grupe musi być początku ukoścni-->
    
    <!--tu będzie body czyli nasza strona -->
    
    </html>    <!--zakączenie grupy html-->


dodatkowe potrzebne informacje

grupa można określić część strony np <head>, <body> i inne ale najważnjejszą funkcją grup jest dodawanie elementów takich jak tekst za pomocą <p> <h1> albo elementów interaktywnych takich jak iframe przyciski i inne ale więcej o tym w innych zakładkach
